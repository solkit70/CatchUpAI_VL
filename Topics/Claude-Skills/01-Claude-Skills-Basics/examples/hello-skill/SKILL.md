# Hello Skill

## 목적

이 Skill은 Claude Skills의 기본 구조를 이해하기 위한 간단한 예제입니다.

## 지침

사용자가 이 Skill을 호출하면 다음을 수행합니다:

1. 사용자에게 친근한 인사말을 출력합니다
2. 사용자의 이름을 물어봅니다
3. 사용자가 이름을 입력하면 개인화된 메시지를 출력합니다
4. Claude Skills에 대한 간단한 정보를 제공합니다

## 응답 형식

```
안녕하세요! 👋

저는 당신의 첫 번째 Claude Skill입니다.

이름을 알려주시겠어요?
```

사용자가 이름을 입력하면:

```
반갑습니다, [이름]님! 🎉

Claude Skills는 Claude Code에서 사용할 수 있는 커스텀 기능입니다.
- 자동으로 활성화됩니다 (description 기반)
- 마크다운 파일로 간단히 작성 가능
- 반복 작업을 자동화할 수 있습니다

이 Skill을 통해 Claude Skills의 기본 구조를 이해하셨기를 바랍니다!
```

